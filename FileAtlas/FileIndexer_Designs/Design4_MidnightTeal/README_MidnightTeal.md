# Design 4 — „Midnight Teal"

| Eigenschaft | Wert |
|-------------|------|
| **Name** | Midnight Teal |
| **Stimmung** | Edel, dunkel, ernst aber elegant — wie ein Finanz-/Analyse-Tool |
| **Farbpalette** | Tiefes Mitternachtsblau (`#0B1220`), Teal/Petrol-Akzent (`#21A89E`), goldene Status-Highlights (`#D9B35C`) |
| **Typografie** | SF Pro, enges Tracking (`-0.2`), kompakte Zeilen, `monospacedDigit` für Zahlen |
| **Sidebar** | Dunkles Glas mit Teal-Tint (`.thin` + getöntes `glassEffect`) |
| **Corner Radius** | 8 pt |
| **Row Height** | 28 pt (**kompakteste** der 5 Designs) |
| **Zielgruppe** | Finanz-/Analyse-Profis, datenintensive Workflows |

## Besonderheit
**Maximale Informationsdichte:** echte Tabelle mit 5 Spalten
(`NAME · KIND · STATUS · SIZE · MODIFIED`), Zebra-Streifen für die Zeilen und
**goldene Status-Badges** für hervorgehobene Zustände (`Final`, `Locked`). Zahlen
nutzen `monospacedDigit`, damit Spalten exakt fluchten.
