# Design 5 — „Frosted Vivid"

| Eigenschaft | Wert |
|-------------|------|
| **Name** | Frosted Vivid |
| **Stimmung** | Modern, lebendig, App-Store-würdig — wie eine Top-Chart macOS App |
| **Farbpalette** | Leuchtendes Purple (`#8C4DF5`) + Electric Blue (`#4D8CFA`) als Glow, Rest neutral-dunkel |
| **Typografie** | SF Pro, **fette** Headings in der Sidebar (`.heavy`), schlankere Body-Schrift in der Liste |
| **Sidebar** | Maximales Frosted Glass (`.ultraThin`) über einem lebendigen Wallpaper-Verlauf, der stark durchscheint |
| **Corner Radius** | 18 pt (sehr rund) |
| **Row Height** | 52 pt |
| **Zielgruppe** | Mainstream-Nutzer:innen, App-Store-Publikum |

## Besonderheit
Die **aktive Auswahl** in der Liste nutzt einen kräftigen Purple→Blue-Verlauf mit
einem weichen **Glow** (farbiger Schatten). Die Sidebar liegt als Frosted-Glass-Panel
über einem bunten Verlaufs-„Wallpaper", damit der Liquid-Glass-Charakter klar sichtbar
wird — auch ohne laufende App im `#Preview`.
