# Design 2 — „Clean Light"

| Eigenschaft | Wert |
|-------------|------|
| **Name** | Clean Light |
| **Stimmung** | Minimalistisch, aufgeräumt, paperlike — wie ein gutes Texteditor-UI |
| **Farbpalette** | Fast nur Weiß & Hellgrau, Akzent in gedämpftem Indigo (`#5C66C7`) |
| **Typografie** | SF Pro (Default), großzügiger Zeilenabstand, subtile Hierarchie |
| **Sidebar** | Maximale Transparenz (`.ultraThin` + klares `glassEffect`), verschmilzt mit dem Hintergrund |
| **Corner Radius** | 12 pt (weich) |
| **Row Height** | 46 pt (viel Luft) |
| **Zielgruppe** | Schreibende, Minimalist:innen, Dokumenten-Verwaltung |

## Besonderheit
**Keine harten Trennlinien** — Struktur entsteht ausschließlich durch Abstände und
Graustufen. Die Auswahl ist nur ein zarter Indigo-Schimmer (10 % Deckkraft), der
Glas-Einsatz in der Sidebar ist bewusst auf ein Minimum reduziert (`.clear`).
