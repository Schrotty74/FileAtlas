# Design 3 — „Warm Studio"

| Eigenschaft | Wert |
|-------------|------|
| **Name** | Warm Studio |
| **Stimmung** | Kreativ, einladend — für Designer:innen & Fotograf:innen mit Asset-Ordnern |
| **Farbpalette** | Warme Töne: Sand-/Off-White (`#F5EEE1`), Orange-Akzent (`#F2851F`) |
| **Typografie** | SF Pro **Rounded**, etwas größere Schrift, weiche Ecken |
| **Sidebar** | Mittleres Glas (`.regular`) mit warmem Orange-Tint |
| **Corner Radius** | 16 pt (sehr weich) |
| **Row Height** | 64 pt (große Thumbnails) |
| **Zielgruppe** | Kreative, Foto-/Design-Workflows |

## Besonderheit
**Vorschau-Thumbnails prominent in der Liste** und großformatig im Detail-Panel
(hier als farbige Gradient-Swatches als Stand-in für echte Bildvorschauen). Datei-
Icons sind deutlich größer dargestellt, jede Zeile hat eine weiche Karten-Optik mit
sanftem Schatten.
